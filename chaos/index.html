How to play
Chaos is a tool to use in conjunction with Magic the Gathering to create crazy interactions and gamestates. This tool can be used with any magic format, and can be modified and used in a numerous amount of ways. In general, it is used as follows. At the beginning of the game, a random effect is rolled on <World>. This has a constant effect on gameplay but can be replaced by other chaos rolls. The game is then continued normally with an added chaos phase before each turn's beginning phase. During this phase, the player rolls on <Chaos> and resolves the effect. The turn then continues normally. This tool can create very weird interactions and it is common to have to decide how an interaction plays out if it is unclear. Have fun!
